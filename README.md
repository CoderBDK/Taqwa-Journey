**Taqwa Journey (তাকওয়া জার্নি)** আপনার প্রতিদিনের আত্মিক উন্নতির এক বিশ্বস্ত মাধ্যম এবং এটি আপনার পরকালীন সফরের এক বিনয়ী সঙ্গী। একজন মুসলিম হিসেবে দৈনন্দিন জীবনে তাকওয়া বজায় রাখা এবং নিজের আমলকে গুছিয়ে নেওয়ার লক্ষ্যেই এই অ্যাপটি তৈরি করা হয়েছে।

---

## 🚀 মূল বৈশিষ্ট্যসমূহ (Features)
* ✨ **ইবাদত ট্র্যাকিং:** সালাত এবং দৈনন্দিন জিকিরের সহজ ট্র্যাকিং।
* 🎯 **লক্ষ্য নির্ধারণ:** আপনার আধ্যাত্মিক উন্নতির জন্য বিশেষ গোল সেট করা।
* 📱 **ইউজার-ফ্রেন্ডলি ডিজাইন:** খুব সাধারণ এবং মার্জিত ইন্টারফেস।
* 🌙 **ডার্ক মোড:** আপনার চোখের আরাম অনুযায়ী ডার্ক থিম পরিবর্তনের সুবিধা।
